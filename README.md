## How to Use

1. Clone this repository:
   ```bash
   git clone https://github.com/IrumShehryar/Javascript-Assignments.git
   ```
2. Open assignment folders for HTML and JS source code.
3. Run the code in your browser or check the live demos for each module.

## Notes

- Each assignment/module is self-contained.
- Points indicate grading/results for submitted coursework.
- Live links require Metropolia’s authentication and server availability.
